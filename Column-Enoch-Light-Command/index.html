    Tony MottolaпјҲгғҲгғӢгғјгғ»гғўгғғгғҲгғ©пјүгҒЁEnoch LightпјҲгӮӨгғҺгғғгӮҜгғ»гғ©гӮӨгғҲпјүгҒҜгҖҒCommand Records гҒ®ж ёеҝғзҡ„гҒӘй–ўдҝӮгҒ§гҒҷгҖӮ

    1950гҖң60е№ҙд»ЈгҒ®гғҸгӮӨгғ•гӮЎгӮӨпјҸгӮ№гғҶгғ¬гӮӘйҢІйҹіж–ҮеҢ–гҒ«гҒҠгҒ„гҒҰгҖҒEnoch LightпјҲгӮӨгғҺгғғгӮҜгғ»гғ©гӮӨгғҲпјү гҒҜвҖңйҹігҒ®е»әзҜүе®¶вҖқгҒЁе‘јгҒ°гӮҢгҒҫгҒ—гҒҹгҖӮгҒқгҒ®еҸіи…•гҒЁгҒ—гҒҰеёёгҒ«жј”еҘҸгҒЁгӮҪгғӯгғҜгғјгӮҜгҒ§ж”ҜгҒҲгҒҹгҒ®гҒҢгҖҒгӮёгғЈгӮәпјҸгӮ№гӮҝгӮёгӮӘгғ»гӮ®гӮҝгғӘгӮ№гғҲгҒ® Tony MottolaпјҲгғҲгғӢгғјгғ»гғўгғғгғҲгғ©пјү гҒ§гҒҷгҖӮCommand RecordsгҒ«гҒҠгҒ‘гӮӢдёЎиҖ…гҒ®й–ўдҝӮгҒҜгҖҒеҚҳгҒӘгӮӢгғ—гғӯгғҮгғҘгғјгӮөгғјгҒЁгғ—гғ¬гӮӨгғӨгғјд»ҘдёҠгҒ®гӮӮгҒ®гҒ§гҒ—гҒҹгҖӮ

    Tony MottolaгҒЁгҒҜ

    гӮўгғЎгғӘгӮ«гҒ®гӮёгғЈгӮәпјҸгӮ№гӮҝгӮёгӮӘгғ»гӮ®гӮҝгғӘгӮ№гғҲпјҲ1918вҖ“2004пјү

    NBCгӮӘгғјгӮұгӮ№гғҲгғ©гӮ„гғҶгғ¬гғ“з•Әзө„гҒ®гӮ»гғғгӮ·гғ§гғігҒ§гӮӮжҙ»иәҚгҒ—гҖҒ1950е№ҙд»ЈгҒӢгӮү70е№ҙд»ЈгҒ«гҒӢгҒ‘гҒҰгғӢгғҘгғјгғЁгғјгӮҜгҒ§жңҖгӮӮеӨҡеҝҷгҒӘгӮ®гӮҝгғӘгӮ№гғҲгҒ®гҒІгҒЁгӮҠгҖӮ

    гӮёгғЈгӮәгҒ гҒ‘гҒ§гҒӘгҒҸгӮӨгғјгӮёгғјгғӘгӮ№гғӢгғігӮ°гҖҒгғ©гғҶгғігҖҒгғқгғғгғ—гӮ№гҖҒжҳ з”»йҹіжҘҪгҒӘгҒ©е№…еәғгҒ„гӮёгғЈгғігғ«гҒ§жј”еҘҸгҖӮ

    Enoch LightгҒЁгҒ®еҮәдјҡгҒ„гҒЁй–ўдҝӮжҖ§

    • Grand Award вҶ’ Command Records жңҹгҒ®еёёйҖЈгӮ®гӮҝгғӘгӮ№гғҲ

    Enoch Light гҒҢGrand AwardгӮ„CommandгӮ’иЁӯз«ӢгҒ—гҒҹйҡӣгҖҒгғӢгғҘгғјгғЁгғјгӮҜгҒ®гғҲгғғгғ—гғ»гӮ»гғғгӮ·гғ§гғійҷЈгӮ’йӣҶгӮҒгҒҫгҒ—гҒҹгҖӮгҒқгҒ®дёӯгҒ§гӮ®гӮҝгғјжӢ…еҪ“гҒ®еёёйҖЈгҒҢTony MottolaгҖӮ

    гҖҢPersuasive PercussionгҖҚгӮ„гҖҢProvocative PercussionгҖҚгӮ·гғӘгғјгӮәгҒ§гӮӮйҮҚиҰҒгҒӘеҪ№еүІгӮ’жһңгҒҹгҒ—гҒҫгҒ—гҒҹгҖӮ

    гғ©гғҶгғіиӘҝгғӘгӮәгғ гӮ„гӮ№гғҶгғ¬гӮӘе®ҡдҪҚгҒ®е®ҹйЁ“гҒ«гҒҠгҒ„гҒҰгҖҒгӮ®гӮҝгғјгҒҜвҖңе·ҰеҸігҒ«жҢҜгӮҠеҲҶгҒ‘гӮ„гҒҷгҒ„жҘҪеҷЁвҖқгҒЁгҒ—гҒҰйҮҚе®қгҒ•гӮҢгҖҒMottolaгҒ®жҹ”и»ҹгҒӘгғ—гғ¬гӮӨгҒҢжҙ»гҒӢгҒ•гӮҢгҒҫгҒ—гҒҹгҖӮ

    • гӮҪгғӯгӮўгғ«гғҗгғ гҒ®гғӘгғӘгғјгӮ№пјҲCommandгҒӢгӮүпјү

    Command RecordsгҒҜгҖҒMottolaгҒ®гӮҪгғӯдҪңе“ҒгӮӮж•°еӨҡгҒҸзҷәиЎЁгҖӮ

    вҖңRoman GuitarвҖқ (1960) гӮ·гғӘгғјгӮәгҒҜзү№гҒ«жңүеҗҚгҒ§гҖҒгғ©гғҶгғіпҪһгӮӨгӮҝгғӘгӮўж°‘и¬ЎгӮ’гғҸгӮӨгғ•гӮЎгӮӨгҒ«гӮўгғ¬гғігӮёгҒ—гҒҹдәәж°—дҪңгҖӮ

    д»ҘйҷҚ вҖңRoman Guitar Vol. 2вҖқвҖңRoman Guitar Vol. 3вҖқ гҒӘгҒ©гҖҒеҪјгҒ®д»ЈиЎЁдҪңгҒҜгҒ»гҒјEnoch LightгҒ®гғ—гғӯгғҮгғҘгғјгӮ№дёӢгҒ§еҲ¶дҪңгҒ•гӮҢгҒҫгҒ—гҒҹгҖӮ

    CommandгҒ®вҖңгғ©гӮҰгғігӮёпјҸгӮӨгғјгӮёгғјгғӘгӮ№гғӢгғігӮ°и·Ҝз·ҡвҖқгӮ’ж”ҜгҒҲгҒҹжҹұгҒ®гҒІгҒЁгҒӨгҒҢMottolaгҒ§гҒ—гҒҹгҖӮ

    з·ЁжӣІгҒЁйҢІйҹігӮ№гӮҝгӮӨгғ«гҒ§гҒ®дҝЎй јй–ўдҝӮ

    • Command RecordsгҒ®еёёйҖЈгӮ®гӮҝгғӘгӮ№гғҲ

    LightгҒҢиЁӯз«ӢгҒ—гҒҹGrand AwardгҖңCommandгҒ«гҒҠгҒ‘гӮӢйҢІйҹігҒ§гҒҜеҝ…гҒҡгҒЁиЁҖгҒЈгҒҰгҒ„гҒ„гҒ»гҒ©еҸӮеҠ гҖӮ вҖңPersuasive PercussionвҖқгӮ·гғӘгғјгӮәгҒӘгҒ©гҒ§гӮ®гӮҝгғјгҒ®е®ҡдҪҚе®ҹйЁ“гӮ’жӢ…гҒЈгҒҹгҖӮ

    • гӮҪгғӯгӮўгғ«гғҗгғ гҒ®гғӘгғӘгғјгӮ№

    CommandгҒӢгӮүгҖҢRoman GuitarгҖҚгӮ·гғӘгғјгӮәгӮ’еұ•й–ӢгҒ—гҖҒгғ©гғҶгғігӮ„гӮӨгӮҝгғӘгӮўйҹіжҘҪгӮ’гғҸгӮӨгғ•гӮЎгӮӨеҢ–гҖӮдё–з•Ңзҡ„гҒ«зҹҘгӮүгӮҢгӮӢд»ЈиЎЁдҪңгҒЁгҒӘгҒЈгҒҹгҖӮ

    • йҹійҹҝе®ҹйЁ“гҒ«гҒҠгҒ‘гӮӢеҪ№еүІ

    гӮ№гғҶгғ¬гӮӘе·ҰеҸігҒ«гӮ®гӮҝгғјгӮ’й…ҚзҪ®гҒҷгӮӢе®ҡдҪҚе®ҹйЁ“

    • гғӘгӮәгғ гӮ»гӮҜгӮ·гғ§гғігҒЁзөЎгӮҖеҚҳйҹігғӘгғјгғү

    гғӘгғҗгғјгғ–гӮ„ж®ӢйҹҝеҮҰзҗҶгҒ®еҠ№жһңгӮ’еј·иӘҝ

    вҶ’ MottolaгҒ®жҹ”и»ҹгҒӘжј”еҘҸгҒҢLightгҒ®гғ“гӮёгғ§гғігӮ’е…·зҸҫеҢ–гҒ—гҒҹгҖӮ

    дё»гҒӘе…ұдҪңгӮўгғ«гғҗгғ дёҖиҰ§

    е№ҙ гӮўгғ«гғҗгғ  и§ЈиӘ¬ гғӘгғігӮҜ
    1959 Persuasive Percussion гӮ®гӮҝгғјгҒ§гӮ№гғҶгғ¬гӮӘе®ҡдҪҚгӮ’жј”еҮәгҖӮеҲқжңҹCommandгҒ®иұЎеҫҙдҪңгҖӮ Amazon
    1960 Roman Guitar CommandгҒӢгӮүгҒ®гӮҪгғӯгғҮгғ“гғҘгғјдҪңгҖӮгғ©гғҶгғіпјҸгӮӨгӮҝгғӘгӮўж°‘и¬ЎгӮ’гғҸгӮӨгғ•гӮЎгӮӨгҒ«гҖӮ Amazon
    1961 Roman Guitar Vol.2 еӨ§гғ’гғғгғҲдҪңгҒ®з¶ҡз·ЁгҖӮгӮЁгӮӯгӮҫгғҒгғғгӮҜгҒӘйҒёжӣІгҖӮ Amazon
    1963 Roman Guitar Vol.3 гӮ·гғӘгғјгӮәе®ҢжҲҗеҪўгҖӮгғ©гӮҰгғігӮёгғ»гӮӨгғјгӮёгғјгғӘгӮ№гғӢгғігӮ°гҒ®йҮ‘еӯ—еЎ”гҖӮ Amazon
    1963 Enoch Light Presents Tony Mottola and the Quad Guitars 4жң¬гҒ®гӮ®гӮҝгғјгӮ’еӨҡйҮҚйҢІйҹігҒҷгӮӢCommandгҒӘгӮүгҒ§гҒҜгҒ®е®ҹйЁ“зӣӨгҖӮ Discogs

    зҸҫеңЁгҒ®зҠ¶жіҒгҒЁеҪұйҹҝ

    Command Records иҮӘдҪ“гҒҜ1970е№ҙй ғгҒ«дәӢе®ҹдёҠгҒ®зөӮз„үгӮ’иҝҺгҒҲгӮӢгҒҢгҖҒдҪңе“ҒзҫӨгҒҜгғӘгӮӨгӮ·гғҘгғјгӮ„й«ҳйҹіиіӘгғӘгғһгӮ№гӮҝгғјгҖҒгӮӘгғјгғҮгӮЈгӮӘгғһгғӢгӮўеҗ‘гҒ‘гҒ®еҶҚзҷәгҒ«гӮҲгӮҠзҸҫеңЁгҒ§гӮӮжөҒйҖҡгҒ—гҒҰгҒ„гӮӢгҖӮPersuasive / Provocative зі»гҒ®жҘҪжӣІгӮ„гӮёгғЈгӮұгғғгғҲгҒҜвҖңгғҹгғғгғүгӮ»гғігғҒгғҘгғӘгғјгғҮгӮ¶гӮӨгғівҖқгӮ„гғҗгғҒгӮ§гғ©гғјгғ‘гғғгғүж–ҮеҢ–гҒ®ж–Үи„ҲгҒ§еј•з”ЁгҒ•гӮҢз¶ҡгҒ‘гҖҒгӮөгғігғ—гғӘгғігӮ°гӮ„гғӘгғҹгғғгӮҜгӮ№гҒ®еҜҫиұЎгҒ«гӮӮгҒӘгҒЈгҒҰгҒ„гӮӢгҖӮзҸҫд»ЈгҒ®гғӘгӮ№гғӢгғігӮ°ж–ҮеҢ–пјҲгғҙгӮЈгғігғҶгғјгӮёгғ»гӮӘгғјгғҮгӮЈгӮӘи¶Је‘ігҖҒгғҗгғјгғҒгғЈгғ«гғ»гғ«гғјгғ гғ»гӮ·гғҹгғҘгғ¬гғјгӮ·гғ§гғігҒӘгҒ©пјүгҒёгҒ®еҪұйҹҝгҒҜе°ҸгҒ•гҒҸгҒӘгҒ„гҖӮ

    гҒҫгҒЁгӮҒпјҡEnoch Light гҒ®йҒәз”Ј

    Enoch LightгҒҜеҚҳгҒ«вҖңгҒ„гҒ„йҹівҖқгӮ’иҝҪжұӮгҒ—гҒҹгҒ гҒ‘гҒ§гҒӘгҒҸгҖҒгғ¬гӮігғјгғүгҒЁгҒ„гҒҶзү©дҪ“гӮ’гғҮгӮ¶гӮӨгғігҒ—гҖҒгӮ№гғҶгғ¬гӮӘжҷӮд»ЈгҒ«гҒҠгҒ‘гӮӢгҖҢиҒҙеҸ–дҪ“йЁ“гҖҚгӮ’еҶҚе®ҡзҫ©гҒ—гҒҹдәәзү©гҒ гҖӮCommand Records гҒ®и«ёдҪңгҒҜгҖҒд»Ҡж—ҘгҒ§гӮӮгӮӘгғјгғҮгӮЈгӮӘгҒ®жӯҙеҸІгҖҒгғҮгӮ¶гӮӨгғіеҸІгҖҒгҒқгҒ—гҒҰгғқгғғгғ—гғ»гӮ«гғ«гғҒгғЈгғјгҒ®ж–Үи„ҲгҒ§еҶҚи©•дҫЎгҒ•гӮҢгҒҰгҒ„гӮӢгҖӮеҲқгӮҒгҒҰиҒҙгҒҸгҒЁгҒҚгҒ®гҖҢй©ҡгҒҚгҖҚгӮ’жј”еҮәгҒҷгӮӢгҒ“гҒЁв”Җв”ҖгҒқгӮҢгҒҢгғ©гӮӨгғҲгҒ®зңҹйӘЁй ӮгҒ§гҒӮгӮҠгҖҒеҪјгҒ®дҪңе“ҒгҒҜзҸҫеңЁгҒ§гӮӮгҒқгҒ®йӯ”еҠӣгӮ’дҝқгҒЈгҒҰгҒ„гӮӢгҖӮ

    гҒҫгҒҹгҖҒEnoch LightгҒЁTony MottolaгҒ®й–ўдҝӮгҒҜгҖҒеҚҳгҒӘгӮӢгҖҢгғ—гғӯгғҮгғҘгғјгӮөгғјгҒЁгӮ®гӮҝгғӘгӮ№гғҲгҖҚгӮ’и¶…гҒҲгҒҰгҒ„гҒҫгҒ—гҒҹгҖӮLightгҒ®йҢІйҹізҫҺеӯҰгҒЁMottolaгҒ®жҹ”и»ҹгҒ§еӨҡеҪ©гҒӘгӮ®гӮҝгғјгғ—гғ¬гӮӨгҒҢзөҗгҒігҒӨгҒ„гҒҹгҒ“гҒЁгҒ§гҖҒCommand RecordsгҒҜгӮ№гғҡгғјгӮ№гӮЁгӮӨгӮёгғ»гғқгғғгғ—гҒ®иұЎеҫҙгҒЁгҒӘгӮҠгҖҒзҸҫеңЁгӮӮдё–з•ҢдёӯгҒ®гӮӘгғјгғҮгӮЈгӮӘгғ•гӮЎгғігӮ’йӯ…дәҶгҒ—з¶ҡгҒ‘гҒҰгҒ„гҒҫгҒҷгҖӮ
