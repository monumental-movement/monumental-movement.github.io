    【コラム】 DAW史を形づくった6社――Apple / Avid / Steinberg / MOTU / Adobe の技術史と市場戦略

    Column 00s 10s 20s DAW
    23 Nov 2025

    序章：DAWの進化は、6つの企業の戦略と技術革新が作りあげた物語である

    文：mmr｜テーマ：Logic／Pro Tools／Steinberg／MOTU／AdobeのDAW技術史

    2025年現在、音楽制作の中心にあるツールは、ほぼ例外なく「DAW（Digital Audio Workstation）」だ。 しかし、この業界が現在の形に至るまでの道筋は単純ではなかった。そこには 企業の買収、技術規格の発明、ハードウェアとソフトの統合、そして 映像・放送との連係 をめぐる40年以上にわたる複雑な歴史がある。

    その全体像を形づくったのは、次の6社である：

    • Apple（Logic Pro / GarageBand）
    • Avid（Pro Tools）
    • Steinberg（Cubase / Nuendo）
    • MOTU（Digital Performer）
    • Adobe（Audition）
    • emagic / Digidesign（買収前）

    これらの企業が取った戦略の違いは、そのまま DAW文化の多極化 を生んだ。 AppleとAvidが買収による“垂直統合”を進めた一方、Steinberg・MOTU・Adobeは“技術規格”や“領域特化”で影響力を確立した。

    1. 1980年代：シーケンサーの時代と「始まりの4社」

    DAW史の起源は、1980年代前半のシーケンサーに遡る。 音声を扱う前の“純粋なMIDI編集ソフト”の時代である。

    ■ Atari ST が生んだ黄金時代

    1985年に登場した Atari ST には 標準でMIDI端子が搭載されていた。 そのため欧州では、次のようなソフトが一気に普及する。

    • Steinberg – Pro 24（後のCubase）
    • C-Lab（後のemagic） – Notator

    Steinberg と emagic は同じドイツ圏で競争しながら発展していった。 特に Steinberg は早期からイベントリスト・ピアノロール・アレンジャーといった今日の基本UIをほぼ揃えていた。

    ■ Macintosh が作曲系ソフトを後押し

    アメリカでは、Macintosh の普及によって MOTU が台頭する。

    • 1985：MOTU「Performer」
    • Macintosh のグラフィカルUIとの相性が良く、多くの作曲家が採用した

    デジタルオーディオ以前、MOTUとSteinbergは作曲家の道具としてスタートした。

    2. 1990年代前半：「DAW」の誕生とオーディオ革命

    90年代に入ると、コンピュータでオーディオを扱えるようになり、DAWの本質が姿を現す。

    ■ Digidesign の衝撃：Pro Tools 1.0（1991）

    • 4トラックデジタル録音
    • 専用DSPによるリアルタイム処理
    • マウスで波形を直接編集できる世界が実現

    Pro Tools が持ち込んだ非線形編集は、もはや“テープの概念”とは違っていた。

    ■ Steinberg の変革：Cubase Audio（1992）

    • MIDI とオーディオの統合
    • 当時としては革新的な編集機能
    • 1996年には VST 1.0 が登場し、DAW史上最大級のインパクトを与える

    この瞬間、ソフトウェアが外部プラグインを受け入れる文化が始まった。

    ■ emagic Logic 1.0（1993）

    • Atari から Mac/Windows へ移行
    • 独創的な「Environment」機能を搭載
    • ソフトシンセを積極的に導入

    emagic と Steinberg はドイツ発の2大DAWとして競合し、 Digidesign は「オーディオ編集の絶対王者」へ向かって進んでいく。

    3. 1990年代後半：三極構造が成立した時代

    1996年、DAW史上最大の転換点が訪れる。

    ■ Steinberg、VST を発表（1996）

    • エフェクトプラグインを外部追加可能に
    • 1999年には VST Instrument（VSTi）でソフトシンセ標準化
    • 音源とエフェクトがPC内で動く文化を作った

    VSTは、後のApple AU、Digidesign RTAS/AAX、Propellerhead Rack Extensionsなど、 すべてのプラグイン規格の“祖先”である。

    ■ MOTU：Digital Performer 2.0 で映像同期を強化

    ハリウッドの作曲家がDPを採用しはじめたのはこの時期。 特に SMPTE同期とテンポマップ編集の精度 が高く評価され、 “映画音楽DAW”としての地位を固めていく。

    ■ Digidesign：Pro Tools|24 でプロスタジオの中心へ

    • 専用DSPを搭載
    • 大規模な録音・編集・ミキシングが可能
    • 映画・放送・音楽スタジオで事実上の標準に

    この頃には、 作曲＝Cubase / Logic / DP 編集・仕上げ＝Pro Tools という役割分担が成立していった。

    4. 2000年代前半：2つの買収劇がDAW史を変える

    ここで、歴史を決定的に変えた“企業買収”が起こる。

    ■ 4-1 Apple が emagic を買収（2002）

    事実として確認されているポイント：

    • 2002年、Apple が emagic を買収
    • Windows版 Logic を終了し、Mac専用化
    • GarageBand（2004）開発へと継承
    • Logic Pro の価格が大きく下がり、普及が進む
    • Apple純正のソフト音源・ループ素材が大幅増強

    この買収は、 「音楽制作＝Mac」のイメージを決定づけた最大要因である。

    ■ 4-2 Avid が Digidesign を統合（1994→2010）

    買収は1994年だが、Digidesignブランドが消滅するのは2010年。

    • Pro Tools は Avid Media Composer と連動
    • 映像編集〜音響仕上げの業界標準の座を完全に奪取
    • 映画制作における「Avid → Pro Tools」という流れが世界的に定着

    Apple と Avid の2社が OSと映像制作の両面からDAW市場の主導権を握る構図 がここで完成する。

    5. 2000–2010年代：Steinberg・MOTU・Adobeの独自戦略

    AppleとAvidが“垂直統合”を進めた一方、 Steinberg・MOTU・Adobeは“専門領域の強化”で勢力を伸ばした。

    ■ Steinberg：Yamaha傘下（2004）

    買収されても方向性は変わらず、技術開発が継続。

    • Nuendo がポストプロダクション市場へ進出
    • VST3（2006）で表現力が拡大
    • Cubase は作曲DAWとして世界的シェアを維持

    ■ MOTU：映画・TV作曲家のスタンダードとして定着

    • DP5/6 時代には映像同期の精度がさらに増強
    • Mac専用であることが映像系と親和
    • ソフトシンセを多用する劇伴作曲家が広く採用

    ■ Adobe：Cool Edit → Audition で放送編集を統合

    買収後はPremiereとの連係を軸に展開。

    • 映像編集者がそのまま音声調整できる
    • ラジオ局・ポッドキャストの定番に
    • ミキサー・作曲DAWというより“編集ツール”の性格が強い

    ここで DAW 市場は、 作曲／録音／編集／映像／放送 の用途ごとに分化していく。

    6. 2010年代：ネイティブ化とOS・ハードウェア統合の時代

    この時期、DAWは CPU依存型ネイティブ処理 が主流になる。

    ■ Apple：Logic Pro X（2013）

    • UIの刷新
    • 大量のソフト音源
    • 価格の低価格化（1万円台へ）
    • Macの性能を最大限に使う最適化

    ■ Avid：HDXとAAXへ移行

    • DSPとネイティブを統合
    • Pro Tools は依然としてスタジオ標準
    • 映画の大規模ミキシングではPro Tools以外の選択肢がほぼ無い状態に

    ■ Steinberg・MOTU・Adobeもアップデートを継続

    • SteinbergはVST3・エンジン刷新
    • MOTUはDP9/10で現代的ワークフローへ
    • AdobeはPodcast需要の増大でAuditionが浸透

    “DAW”という言葉はこの頃には完全に一般化され、 宅録文化の拡大とも相まって市場全体が急速に拡大する。

    7. 2020–2025：Apple Siliconとサブスク時代

    直近数年間、DAW史はまた大きな変化を迎えている。

    ■ Apple：Apple Silicon による圧倒的高速化

    • Logic Pro はARMアーキテクチャに最適化
    • iPad版 Logic（2023）でモバイル制作を牽引
    • 低レイテンシと高効率が高く評価される

    ■ Avid：Pro Tools のサブスク移行

    • 永続ライセンスからサブスクへ
    • Dolby Atmos ワークフローが標準化
    • 映画音響市場での支配力は継続

    ■ Steinberg：Nuendo のハイエンド領域強化

    • Dolby Atmos対応
    • ゲームオーディオ業界でも存在感が増す

    ■ MOTU：DP11で新時代へ

    • 作曲〜映像同期ワークフローを強化
    • インスト作曲家・劇伴作家の支持が続く

    ■ Adobe：配信・放送需要の増加

    • Premiereとの統合優位性
    • YouTube・Podcast時代に強い

    8. 6社比較：DAW史における“役割”の違い

    DAW史を支える6社は、明確な役割分担が形成されている。

    企業 主導領域 意味
    Apple（Logic） 作曲・宅録 Macと統合、ソフト音源文化を拡大
    Avid（Pro Tools） 映画・音響・スタジオ 専用DSPと編集能力、映像連係
    Steinberg 規格化（VST）・総合DAW 全DAWに影響を与えた技術基盤
    MOTU 映像同期・劇伴作曲 フィルムスコアリングの標準
    Adobe 放送・編集・配信 Premiere統合による映像制作向け
    emagic / Digidesign 買収前の源流 Mac専用化／映画音響の基盤形成

    9. 年表（Mermaid）

    ■ DAW史総覧（1980–2025）

    timeline title DAW史総覧：Apple / Avid / Steinberg / MOTU / Adobe 1984 : Steinberg & Digidesign 設立 1985 : MOTU Performer 1989 : Cubase 1.0 1991 : Pro Tools 1.0 1993 : Logic 1.0 1996 : VST 1.0 1999 : VSTi 2002 : Apple が emagic を買収（Logic Mac専用に） 2003 : Adobe が Cool Edit を買収（Audition誕生） 2004 : Steinberg が Yamaha 傘下へ 2010 : Digidesign ブランド終了 2013 : Logic Pro X 2020 : Pro Tools サブスク中心へ 2023 : Logic Pro for iPad

    10. 結語：DAW史は「企業戦略 × 技術規格 × 市場ニーズ」が交わる巨大な物語

    Apple が emagic を買収し、Avid が Digidesign を統合したことで、 DAW市場は OS × 映像制作 × スタジオインフラ の軸で大きく変わった。 一方で、Steinberg は VST という歴史的規格を創り、 MOTU は 映像同期 を極め、 Adobe は 放送・動画編集との統合 で独自の市場を形成した。

    6社は互いに競争しつつも、 結果として「音楽制作という文化そのもの」をともに形づくってきた。

    2025年以降、AI編集、クラウド共同制作、モバイル制作の拡大など、 DAWの未来はさらに新しい局面を迎えるだろう。 しかし、その根底には必ず、ここで述べた6つの企業の足跡がある。
