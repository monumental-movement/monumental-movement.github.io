    【專欄】為什麼AKIRA聽起來仍然像是未來的聲音？ — 前衛民謠與賽博龐克的復興

    Column zh-hant 80s Anime Cyberpunk FilmMusic
    27 Feb 2026

    前言：為什麼“AKIRA 聽起來仍然像是未來”

    文：mmr｜主題：為什麼AKIRA聽起來還是「未來之聲」——前衛民謠所描繪的關於城市和身體的預言

    1988年，正值泡沫經濟時期的日本上映了一部動畫電影。原作是大友克洋的作品。音樂由娛樂山城組的領導者山城正司創作。作品名稱為《AKIRA》。

    儘管上映已近40年，這部電影的聲音卻從未過時。事實上，近年來，隨著賽博龐克的復興，《Why AKIRA Still Sounds Future》的脈絡變得更加濃厚。

    原因不僅僅是懷舊。失控的城市、國家與科技之間的緊張關係、年輕人的爆發、身體的轉變。配樂融合了民間音樂和電子音樂，與當今世界產生了奇異的共鳴。

    在本文中，我們將從事實出發，從AKIRA的音樂的製作背景、聲學結構、文化影響以及重新評價的過程來解讀AKIRA的音樂如何成為「未來之聲」。

    AKIRA 的聲音預示著我們的現在，而不是 1988 年的未來。

    城市預言：新東京的聲學空間

    原作及電影化過程

    原作漫畫《AKIRA》於1982年開始連載，故事發生在核戰後的新東京。電影版於1988年上映，總製作成本在當時是空前的，成為日本動畫史上的傑出項目。

    故事發生在2019年的新東京。軍事實驗、反政府示威、摩托車幫、超自然力量的發展。它描繪了一座城市的崩潰與重生，強烈反映了冷戰結束時的氣氛。

    音樂的構思與電影的製作同時進行。大友導演尋求一種更基本的宗教聲音，而不是現有的類似動漫的伴奏。結果，被挑出來的就是娛樂山城組。

    與賽博朋克的位置關係

    AKIRA 經常被評為賽博龐克的傑作。 《銀翼殺手》於 1982 年上映，比電影上映早了六年。這首曲子使用 Vangelis 的合成聲音描繪了城市的頹廢。

    另一方面，AKIRA並不僅僅用電子聲音來表達未來。相反，它同時凸顯了民族、儀式和物理聲音、發聲技術和魔法。

    這個選擇後來更新了賽博朋克的形象。

    flowchart LR A[1982 原作連載開始] --> B[1988 映画公開] B --> C[前衛民族サウンド採用] C --> D[都市×儀式的音響] D --> E[再評価と再燃]

    新東京不僅是透過視覺建造的，還透過聲音建造的。

    前衛民族聲音的結構：娛樂山城組的挑戰

    藝能山城組是？

    藝能山城組是以山城正司為中心組成的研究和實踐民間音樂的音樂團體。他的研究主題包括印尼加美蘭、峇裡島儀式音樂和非洲多元節奏音樂，並試圖以自己的方式重建它們。

    在製作AKIRA的音樂時，他們應用了實際的聲樂技巧和打擊樂器結構，而不是簡單地創造「民間音樂風格」。

    片頭曲《Kaneda》具有像徵意義。鼓、合唱和交際發聲。節拍不是簡單的四拍節拍，而是層次豐富的複雜節奏。

    電子與民族聲音的融合

    AKIRA 的配樂結合了民族元素和合成器。然而，電子聲音並不是主角。它僅起到空間擴展裝置的作用。

    這種結構創造未來感的原因是顯而易見的。

    20 世紀 80 年代的「未來聲音」大部分都以合成器為中心。然而，AKIRA 將可以用原始來形容的聲音和打擊樂器帶到了最前沿。結果是無法確定時間軸的聲音。

    這是一種既不屬於過去也不屬於未來的聲音。

    flowchart TD A[民族的打楽器] --> D[音響的重層構造] B[合唱・声明] --> D C[シンセサイザー] --> D D --> E[時間軸を超えた未来感]

    AKIRA 的未來不僅建立在技術上，還建立在儀式的聲音上。

    身體與聲音：逃亡與變形的聲音設計

    自行車場景的節奏

    電影開頭的摩托車追逐不僅僅是一個動作場面。對節奏的強調、與引擎聲音的同步以及鼓點的敲擊創造了身體上的興奮。

    聲音不跟隨視覺，而是代表身體的心跳。觀眾下意識跟上節奏。

    Tetsuo的變身與聲音

    在故事的後半部分，鐵男的猖獗能力和身體變化透過聲音得到了有力的描繪。高頻電子音、起伏的低音和緊張的副歌相結合，營造出一種聽得見的不安感。

    這裡重要的是聲音不僅是表演，而且是「故事的一部分」。

    音樂不是背景。它作為一種聲音而存在，象徵著城市的意志、國家的壓力和個人的崩潰。

    AKIRA的聲音並不是角色的情緒，而是這座城市本身的心跳。

    年表：AKIRA 與賽博龐克的交集

    timeline 1982 : 原作連載開始 1988 : 映画公開 1990s : 海外でカルト的人気拡大 2000s : デジタルリマスター・再上映 2020 : サイバーパンク再評価の波

    1988年上映後，不僅在日本上映，還在海外上映，在北美和歐洲獲得了狂熱追捧。

    進入 21 世紀，藍光和 4K 重製版正在重新評估它。

    此外，近年來遊戲和動漫的復興也強化了賽博龐克的脈絡。其中的標誌之一是《Cyber​​punk：邊緣奔跑者》。

    この再燃の中で、AKIRAは「元祖」ではなく、「完成形」として再び参照されている。

    《AKIRA》並不是一部走在時代前沿的作品，而是一部不斷挑戰時代的作品。

    影響力與傳承：傳播到全球音樂和視覺文化

    AKIRA 的視覺效果和音樂影響了許多創作者。霓虹燈城市、摩托車、軍事研究和離家出走的年輕人。

    在音樂方面，民族元素和電子聲音融合的想法後來被連接到實驗聲音和俱樂部音樂。

    尤其重要的是，它打破了「未來=冷酷的電子聲音」的刻板印象。相反，它表明未來可以透過物理和本土的共鳴來描繪。

    《AKIRA》對世界的影響不是作為一句名言，而是作為思想的更新。

    為什麼現在又發生了？

    真正的城市已經迎頭趕上

    巧合的是，2019年是AKIRA的舞台年。流行病、社會動盪、國家和個人緊張局勢。似乎真實的城市已經追上了小說。

    賽博朋克的回歸

    賽博龐克美學在遊戲、動漫和時尚中正在被重新評估。霓虹燈、義肢和監視社會。 AKIRA 是這個視覺代碼的起源。

    但比視覺更重要的是聲音。

    在數位時代，多層的實體聲音和打擊樂聽起來矛盾地新鮮。

    AKIRA 的聲音不是由人工智慧或演算法創造的，而是由人類集體發聲和呼吸創造的。

    在這個數位時代，AKIRA 的實體聲音聽起來就像未來。

    結論：繼續敲響未來鐘聲的作品

    AKIRA 的音樂並非固定在特定時代。 1988年的實驗精神，民族音樂研究的成果，動漫史的轉捩點。它們已經結晶，並且仍然以新的方式產生共鳴。

    「Why AKIRA Still Sounds Future」という問いへの答えは単純だ。

    未来を電子音で描かなかったからだ。

    儀式、合唱、打擊樂和城市噪音。透過混合它們，他創造了一種不受時間軸限制的聲音。

    AKIRA 沒有預測未來。它呈現了未來的聲音。

    AKIRAは終わった作品ではなく、いまも鳴り続ける未来そのものだ。
