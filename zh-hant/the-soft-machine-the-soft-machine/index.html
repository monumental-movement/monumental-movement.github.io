    The Soft Machine 是一支來自坎特伯雷的英國前衛爵士搖滾樂隊，由 Mike Rutledge、Robert Wyatt、Kevin Ayers、David Allen 和 Larry Nowlin 於 1966 年中期組建。該樂隊以威廉·S·巴勒斯的小說“The Soft Machine”命名。
