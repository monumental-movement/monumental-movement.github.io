    【專欄】Vaportrap/Cloud Rap的歷史和發展

    Column zh-hant Cloudrap Hiphop Vaportrap
    06 Dec 2025

    序言：什麼是“雲說唱”？

    文：mmr｜主題：關於與Lo-Fi Hip Hop相交的雲說唱譜系

    Cloud Rap 興起於 2010 年左右，以灣區和美國地下互聯網為中心，在與後蒸汽波亞文化美學、Chopped & Screwed、Lo-Fi Hip-Hop 和現代情緒說唱相結合的同時發展。特別是，Lil B、Clams Casino 和 Main Attrakionz 的活動象徵著早期，形成了後來形成 Lil Peep、XXXTENTACION、Bones 和 Yung Lean 的血統基礎。

    Cloud Rap 遠比傳統 Hip Hop 更加環境化，特點是更環境化的聲音、強烈混響的人聲處理、帶有空間 FX 的夢幻質感，後來演變為 Vaportrap，吸收了電子音樂和類似 Vaporwave 的 meme 文化。

    一、Cloud Rap之前的準備步驟（2000-2008）

    ● 源自互聯網的嘻哈和混音帶文化

    2000 年代末，MySpace 和 DatPiff 擴展了互聯網發行的混音帶文化，允許不受地域限制的協作。這成為了 Cloud Rap“從地理中解放出來”的土壤。

    ● 切斷和擰緊的影響

    Screw處理（放慢速度、改變音調、強調混響），源自DJ Screw的方法，對後來Cloud Rap的“空靈節奏”做出了巨大貢獻。

    二. Cloud Rap的組建（2009-2012）

    ●Lil B和Based World文化區

    2009年左右，Lil B發展出了一種與網絡文化高度契合的說唱風格，支持他音樂的製作人就形成了Cloud Rap的雛形。

    ● Clams 賭場簡介

    Clams Casino 的器樂合集於 2011 年左右發布，他們為 A$AP Rocky 提供的音樂以其“迷霧墊”、“環境旋律”和“深度空間處理”決定了 Cloud Rap 的聲音形象。

    ● 主要景點及場景擴展

    Main Attrakionz 將“雲”一詞確立為自己的風格，擴大了 Bandcamp 一代藝術家的非商業、DIY 文化領域。

    三． Vaportrap 的出現（2013-2016）

    Vaportrap 是一種將 Vaporwave 的方法論（採樣、重建 80-90 年代消費文化、環境紋理）與 trap 的節拍結構相結合的風格。

    ● 審美特徵

    • 90 年代消費文化的質感（CM、深夜秀、虛擬空間等）
    • 使用極端混響和延遲進行空間處理
    • BPM 與 Trap (120-150) 幾乎相同，但質感與 Vaporwave 相似。

    ● 深化生產者文化

    以SoundCloud為中心，在個人家庭錄音層面發生高速製作和高速消費，形成了從Cloud Rap到Vaportrap的流程。

    四． Lo-Fi HipHop 與“Cloud Rap”的聯繫

    ● Lo-Fi HipHop 方法論

    • 採樣器文化，例如 MPC
    • 噪聲、模擬紋理、盒式磁帶失真
    • 隨著 Chillhop/Study Beats 的發展，流媒體文化不斷擴展

    Cloud Rap 的人聲處理輕鬆地與 Lo-Fi Hip Hop 混合，Lil Peep 和 XXXTENTACION 的作品結合了吉他聲音 x Lo-Fi 處理 x Cloud Rap 般的漂浮感。

    五、情緒說唱/悲傷說唱的興起（2016-2018）

    ● Lil Peep 的影響力

    Lil Peep 發展了一種混合了情緒、朋克和雲說唱元素的風格，形成了後 SoundCloud Rap 美學的核心。尤其是吉他樣本x混響x悲傷旋律的組合，對後來的歌曲產生了很大的影響。

    ● XXXTENTACION的多樣性

    XXXTENTACION 練習了多種風格，從雲說唱到硬核、R&B 和另類。最小的聲音結構、對低頻的強調以及環境元素都符合 Vaportrap 的方向。

    六．現代 Vaportrap/雲說唱 (2019–2025)

    ● 國際化，消除地區差異

    隨著 YouTube、SoundCloud 和 TikTok 的興起，Cloud Rap 技術正在傳播到世界各地。類似的生產方法在亞洲和歐洲已經很常見。

    ● 生成式 AI 帶來節拍文化的變化（基於事實）

    在 2020 年代末，使用擴散和生成模型的雲狀墊和環境紋理生成已經變得司空見慣，使得家庭環境中的生產速度更快。

    聲音結構分析

    ● Vaportrap/Cloud Rap 的典型節拍元素

    • Pad：深度混響合成器、環境和弦
    • 鼓：Trap（808、踩镲、深踢）
    • BPM：120–150
    • 人聲處理：更深的混響、自動調音、配音處理
    • 質地：90 年代/2000 年代文化名言

    結論：“雲說唱”將走向何方？

    Cloud Rap 不僅僅是一種流派，它已經成為後互聯網嘻哈製作方式、美學和發行文化的象徵。與 Vaportrap 的融合繼續表明音樂圍繞在線審美領域而不是物理環境的方向。

    未來，通過與AI世代聯動，預計Cloud Rap般的質感將變得更加抽象，擴展為更加環境化和“非人性”的聲像。
