    【專欄】Butthole Surfers：美國迷幻朋克特立獨行者的完整歷史

    Column zh-hant 80s 90s Alternative Psychedelic Punk
    22 Nov 2025

    **序幕：從美國南部出現的<混沌裝置>**

    文：mmr｜主題：關於Butthole Surfers的音源、現場表演、錄音技巧、製作、設備

    在 20 世紀 80 年代的美國地下組織中，很少有人能像 Butthole Surfers 那樣怪異、暴力和充滿幻想。 他們的現場表演經常被描述為“震撼搖滾”、“重迷幻”和“混亂”，但他們的本質要復雜得多。

    *混亂是由計算造成的， 不和諧音是由即興創作產生的， 迷幻是結構化的。 *

    1.從形成到早期活動（1981-1983）

    該樂隊在德克薩斯州組建，從一開始就擁有與它同義的元素：吉他和薩克斯管的刺耳聲音、雙鼓、反饋、閃光燈和電影放映。

    雖然成員隨著時間的推移而變化，但最初的核心成員如下。

    • 吉比·海恩斯 — 聲樂、吉他、視覺指導
    • 保羅·利裡 — 吉他
    • King Coffey — 鼓
    • Teresa Taylor — 鼓（雙鼓配置）

    尤其是吉比·海恩斯和保羅·利裡，儘管音樂取向不同，但他們找到了和諧，並開始了一項類似於藝術團體的活動。

    2.早期EP“Butthole Surfers”（1983）

    樂隊的第一個聲音混合了晚期朋克/硬核的快速與迷幻的無定形性，具有以下特徵：

    ● 曲別・事実ベース分析

    The Shah Sleeps in Lee Harvey’s Grave

    • 吉他主要使用直接錄製的反饋
    • 薩克斯管演奏不規則，破壞了旋律。
    • 歌詞碎片化，表演感很強。

    嗅嗅女士

    • 模擬噪聲和語音處理
    • 不使用採樣器，處理是通過工作室效果完成的

    ● 記錄方式（圖）

    流程圖TD A[“直接吉他錄音”] –> B[“失真處理”] B –> C[“薩克斯管即興錄音”] C –> D[“鼓奇數拍號 1 錄音”] D –> E[“整體混音（Lo-Fi）”]

    3.八十年代的混亂時期：通靈……無能為力……另一個男人的囊（1984）

    當時的現場佈景是由真人16mm鏡頭、頻閃燈和噪音吉他組合而成，營造出一個“儀式感”的空間，考驗觀眾的承受能力。

    ● 代表歌曲

    • 酒吧燒烤教皇
    • 墓地
    • 牛仔鮑勃

    ● 特點

    • 許多歌曲都有奇怪的拍號
    • 吉他和薩克斯管的刺耳聲音
    • 錄音為模擬 4-8 軌

    流程圖TD A[“多軌錄音”] –> B[“吉他/薩克斯管效果”] B –> C[“鼓奇數拍號錄音”] C –> D[“添加即興部分”] D –> E[“混音/母帶處理（Lo-Fi）”]

    4.倫勃朗小貓 (1986)：加深迷幻

    這是工作室工作急劇發展的時期。 這張專輯的設計目的是在整個專輯中給人一種“模糊的焦慮感”，包括多層的鼓、反饋、倒置的聲音以及不同的聲場結構。

    流程圖TD A[“吉他連复段/噪音”] –> B[“多層鼓錄音”] B –> C[“薩克斯管即興創作”] C –> D[“立體聲場調整”] D –> E[“混音和母帶處理”]

    5. Hairway to Steven (1988)：更長、更有條理

    長歌和短歌交替排列，“歌曲的輪廓”變得比以前更加明顯。 為了提高現場再現性，錄音室錄音也進行了明確的角色劃分。

    流程圖TD A[“錄製一首長歌”] –> B[“錄製一首短歌”] B –> C[“薩克斯管/聲音變化”] C –> D[“噪聲堆棧”] D –> E[“舞台混音”]

    6。人聲/聲音結構穩定的 90 年代

    獨立蠕蟲沙龍 (1993)

    進入大廠牌時代，無論是音質還是表現都變得更加清晰。

    流程圖TD A[“吉他/貝斯/鼓單獨錄音”] –> B[“效果處理”] B –> C[“薩克斯管即興錄音”] C –> D[“混合立體聲調整”]

    電動樂園 (1996)

    包括簽名歌曲Pepper。 雖然旋律被凸顯出來，但噪音和陌生感仍然存在。

    流程圖TD A[“旋律聚焦錄音”] –> B[“吉他/薩克斯管處理”] B –> C[“節奏模式優化”] C –> D[“噪音/迷幻效果調整”] D –> E[“混音母帶”]

    7.怪異革命 (2001)：電子聲音引入時期

    在這張專輯中，採樣器數字效果被認真引入，樂隊的聲音形象發生了巨大的變化。

    7. 專輯所有歌曲及錄音方法詳細解說

    7.1 屁眼衝浪者EP (1983)

    曲目列表和功能

    1. 國王睡在李·哈維的墳墓裡

      • 直接錄製的吉他即興演奏，分層失真
      • 薩克斯管用簡短的樂句形成刺耳的聲音
      • 鼓使用了很多奇怪的拍號，在錄音室裡他們基本上是一次性錄製的。

    2. 嗅嗅女士

      • 以噪聲處理為中心的短曲
      • 使用模擬設備即興創作，無需使用樣本或磁帶循環
      • 表演會根據觀眾的反應而變化。

    錄音方式

    流程圖TD A[“直接吉他錄音”] –> B[“失真處理”] B –> C[“薩克斯管即興錄音”] C –> D[“鼓奇數拍號 1 錄音”] D –> E[“整體混音（Lo-Fi）”]

    7.2 通靈……無能為力……另一個人的囊（1984）

    歌曲示例

    • Bar-B-Q Pope – 吉他和薩克斯管的刺耳聲音，支離破碎的歌詞
    • 墓地 – 鼓奇怪的拍號和噪音，朋克線性
    • 牛仔鮑勃 – 迷幻與幽默，現場可複制的結構

    錄音方式

    流程圖TD A[“多軌錄音”] –> B[“吉他/薩克斯管效果”] B –> C[“鼓奇數拍號錄音”] C –> D[“添加即興部分”] D –> E[“混音/母帶處理（Lo-Fi）”]

    7.3 倫勃朗小貓 (1986)

    • 複雜的節奏結構、吉他雜音和反饋放大
    • 薩克斯管在整首歌中繼續即興演奏。
    • 在錄音室中，鼓聲被多層錄製以強調聲場的深度。

    流程圖TD A[“吉他連复段/噪音”] –> B[“多層鼓錄音”] B –> C[“薩克斯管即興創作”] C –> D[“立體聲場調整”] D –> E[“混音和母帶處理”]

    7.4 史蒂文的發路 (1988)

    • 迷幻與哥特式的融合
    • 更改每首歌曲的錄音風格（長歌和短歌的比較）
    • 保留實驗部分，同時強調現場再現性

    流程圖TD A[“錄製一首長歌”] –> B[“錄製一首短歌”] B –> C[“薩克斯管/聲音變化”] C –> D[“噪聲堆棧”] D –> E[“舞台混音”]

    7.5 獨立蠕蟲沙龍（1993）

    • 提高錄音質量，與首次亮相保持一致
    • 使用錄音室多軌並準備歌曲結構
    • 單獨錄製吉他、貝斯、鼓和薩克斯管，以精確控制噪音和效果

    流程圖TD A[“吉他/貝斯/鼓單獨錄音”] –> B[“效果處理”] B –> C[“薩克斯管即興錄音”] C –> D[“混合立體聲調整”]

    7.6 電力樂園 (1996)

    • 包含 MTV 熱門歌曲 Pepper
    • 噪音和迷幻元素繼續存在，同時強調流行
    • 將人聲旋律放在最前面的混音方法
    • 錄音設計可最大限度提高現場保真度

    流程圖TD A[“旋律聚焦錄音”] –> B[“吉他/薩克斯管處理”] B –> C[“節奏模式優化”] C –> D[“噪音/迷幻效果調整”] D –> E[“混音母帶”]

    7.7 奇怪的革命（2001）

    • 推出電子聲音和採樣器
    • 與傳統模擬聲音的融合
    • 遵守主要標籤標準，同時保持實驗性

    流程圖TD A[“電子音/採樣器介紹”] –> B[“模擬樂器融合”] B –> C[“吉他/薩克斯管即興創作”] C –> D[“鼓模式多層錄音”] D –> E[“最終混音/母帶處理”]

    8.現場巡演歷史和視覺製作的完整結構

    Butthole Surfers 最重要的元素是現場。 僅根據事實，可以將生產系統化如下。

    8.1 1980 年代：衝擊搖滾現場的起源

    • 射出頻閃燈
    • 使用 16 毫米膠片的抽像圖像
    • PA 調音台強調吉他反饋
    • 雙鼓位於舞台中央的配置。
    • 薩克斯管不協和即興演奏

    流程圖LR A[“舞台中央：雙鼓”] –> B[“吉他（反饋）”] A –> C[“薩克斯管（即興創作/不協和音）”] B –> D[“頻閃燈”] C –> E[“16mm 膠片投影”] D –> F[“PA 台：混響/延遲增強”]

    8.2 1990–1993：心理擴展

    流程圖TD A[“吉他：高輸出放大器”] –> B[“噪聲層”] B –> C[“PA: 中高頻範圍增強”] C –> D[“後台：視頻投影”] D –> E[“觀眾體驗：沉浸式迷幻空間”]

    8.3 電子聲音引入時期（2001年–）

    流程圖TD A[“電子聲音/採樣器”] –> B[“實時 PA 同步”] B –> C[“視頻：數字投影”] C –> D[“同步樂隊表演”] D –> E[“高分辨率照明系統”]

    8.4 2000年代：引入電子聲音和採樣器之後

    ● 特點

    • 現場實現“Weird Revolution”的電子聲音
    • 通過再現合成器採樣板樣本來改變舞台聲像
    • 視頻製作已轉向數字化，提高了同步性

    ● 現場演出地圖（2001年–）

    流程圖TD A[“電子聲音/採樣器”] –> B[“實時 PA 同步”] B –> C[“視頻：數字投影”] C –> D[“同步樂隊表演”] D –> E[“高分辨率照明系統”]

    9. Butthole Surfers 使用的設備列表

    多年來，Butthole Surfers 使用的設備變化很大，但文獻中保留了以下內容。

    9.1 吉他/擴音器

    • 吉布森SG
    • Fender Stratocaster
    • Fender Twin Reverb（高輸出）
    • Peavey 放大器（高抗噪性）

    9.2 基礎

    • Fender 精密貝斯
    • Ampeg SVT（實時且高概率）

    9.3 鼓

    • Tama / Ludwig（均為鼓）
    • 雙鼓組合（King Kofi 和 Teresa Taylor）

    9.4 薩克斯管

    • 中音薩克斯管（主要是即興演奏）
    • 向 PA 添加延遲/混響處理

    9.5 電子聲音（2000年代）

    • 採樣器：AKAI型號
    • 合成器墊：Roland
    • 數字效果：Lexicon

    10.所有活動的時間順序（基於事實的簡化版本）

    流程圖TD A[“1981：在德克薩斯州成立”] –> B[“1983：第一張 EP 發行”] B –> C[“1984–1986：擴大北美巡演”] C –> D[“1987–1989：歐洲擴張和迷幻時期”] D –> E[“1993：過渡到主要時期”] E –> F[“1996:電氣界時期”] F –> G[“2001：電子聲音的介紹”] G –> H[“之後：繼續進行不規則活動”]

    11.摘要：《Butthole Surfers》的音樂意義

    Butthole Surfers 的音樂經常被描述為“怪異”和“瘋狂”。 然而，它的核心是一個結構實驗。

    • 不和諧不是即興的，而是“有意的安排”
    • 噪音是聲學設計的一部分
    • 現場表演的混亂在於燈光、視頻和擴聲的精確協調。
    • 迷幻的構建就像一個故事

    結果，他們成為美國地下音樂的重要支柱，並影響了另類搖滾的基礎。

    Butthole Surfers 繼續呈現音樂形式的“外部”。 它仍然在朋克、噪音搖滾和迷幻音樂的交匯處佔據著獨特的地位。
