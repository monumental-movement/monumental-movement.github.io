    【專欄】塑造 DAW 歷史的六家公司：Apple / Avid / Steinberg / MOTU / Adob​​​​e 技術歷史和市場策略

    Column zh-hant 00s 10s 20s DAW
    23 Nov 2025

    序言：DAW的演變是一個由六家公司的戰略和技術創新創造的故事。

    文：mmr｜主題：Logic/Pro Tools/Steinberg/MOTU/Adobe DAW 技術史

    截至 2025 年，音樂製作的核心工具幾乎無一例外都是 DAW（數字音頻工作站）。 然而，該行業形成目前形態的道路並不簡單。它有著複雜的歷史，跨越 40 多年的企業收購、發明技術標準、集成硬件和軟件以及鏈接視頻和廣播。

    以下六家公司塑造了整體格局：

    • 蘋果（Logic Pro/GarageBand）
    • Avid（專業工具）
    • 斯坦伯格（Cubase/Nuendo）
    • MOTU（數字表演者）
    • Adobe（試鏡）
    • emagic/Digidesign（收購前）

    這些公司採取的策略的差異導致了DAW文化的多極化。 蘋果和Avid通過收購追求“垂直整合”，而斯坦伯格、MOTU和Adobe則通過“技術標準”和“領域專業化”建立影響力。

    1. 20 世紀 80 年代：測序儀時代和“開創它的四家公司”

    DAW 歷史的起源可以追溯到 20 世紀 80 年代初的音序器。 這是“純MIDI編輯軟件”在處理音頻之前的時代。

    ■ Atari ST誕生的黃金時代

    Atari ST 於 1985 年推出，標準配備了 MIDI 終端。 結果，以下軟件很快在歐洲流行起來。

    • Steinberg – Pro 24（後來的 Cubase）
    • C-Lab（後來的 emagic） – Notator

    Steinberg 和 emagic 在同一德國地區競爭時發展起來。 特別是，Steinberg 很早就擁有了當今大部分的基本 UI，如事件列表、鋼琴捲簾、編曲器等。

    ■ Macintosh 增強了作曲軟件

    在美國，Macintosh 的普及導致了 MOTU 的興起。

    • 1985：MOTU“表演者”
    • 它與 Macintosh 圖形用戶界面兼容，並已被許多作曲家採用。

    在數字音頻出現之前，MOTU 和 Steinberg 最初是作為作曲家的工具。

    2. 20 世紀 90 年代初期：“DAW”的誕生和音頻革命

    20 世紀 90 年代，計算機開始能夠處理音頻，DAW 的真正本質也隨之顯現。

    ■ Digidesign 影響：Pro Tools 1.0 (1991)

    • 4軌數字錄音
    • 專用DSP實時處理
    • 實現了用鼠標直接編輯波形的世界

    Pro Tools 引入的非線性編輯不再是“磁帶的概念”。

    ■ 斯坦伯格的轉變：Cubase Audio (1992)

    • MIDI 和音頻集成
    • 當時創新的編輯功能
    • VST 1.0 於 1996 年推出，是 DAW 歷史上影響最大的版本之一。

    此時，一種軟件接受外部插件的文化開始了。

    ■ emagic Logic 1.0 (1993)

    • 從 Atari 遷移到 Mac/Windows
    • 配備獨特的“環境”功能
    • 積極引入軟合成器

    emagic 和 Steinberg 作為德國兩大 DAW 相互競爭。 Digidesign正朝著成為“音頻編輯的絕對王者”邁進。

    3、20世紀90年代後期：三極格局確立的時代

    1996年，DAW歷史上最大的轉折點發生了。

    ■ Steinberg 宣布推出 VST (1996)

    • 可以外部添加效果插件
    • 1999 年，VST Instrument (VSTi) 被標準化為軟合成器。
    • 創建了一種在 PC 上運行聲源和效果的文化

    VST後來成為Apple AU、Digidesign RTAS/AAX、Propellerhead Rack Extensions等。 它是所有插件標準的“祖先”。

    ■ MOTU：與 Digital Performer 2.0 增強視頻同步

    正是在這個時期，好萊塢作曲家開始使用DP。 特別是，SMPTE 同步和速度圖編輯準確性受到高度讚揚。 鞏固其“電影音樂 DAW”的地位。

    ■ Digidesign：將 Pro Tools|24 引入您專業工作室的核心

    • 配備專用DSP
    • 可進行大規模錄音、編輯和混音
    • 電影、廣播和音樂工作室的事實標準

    到了這個時候， 作曲= Cubase / Logic / DP ** **編輯/整理 = Pro Tools 建立了角色分工。

    4. 2000 年代初期：兩次收購改變了 DAW 的歷史

    至此，一場改變歷史的“企業收購”果斷發生了。

    ■ 4-1 蘋果收購 emagic (2002)

    已確認為事實的要點：

    • 蘋果公司於 2002 年收購了 emagic
    • 終止 Logic 的 Windows 版本並將其專門用於 Mac
    • 繼承 GarageBand (2004) 開發
    • Logic Pro 的價格大幅下降，而且受歡迎程度不斷提高。
    • 蘋果正版軟件音源和循環素材大幅增加

    此次收購是 這是決定“音樂製作=Mac”形象的最大因素。

    ■ 4-2 Avid 整合 Digidesign (1994→2010)

    此次收購發生於 1994 年，但 Digidesign 品牌於 2010 年消失。

    • Pro Tools 與 Avid Media Composer 配合使用
    • 完全接管視頻編輯和聲音處理的行業標準
    • 電影製作中“Avid → Pro Tools”的趨勢已在全球範圍內確立。

    兩家公司：Apple 和 Avid 計劃在操作系統和視頻製作方面佔據DAW市場的領先地位 到這裡就完成了。

    5. 2000-2010年代：Steinberg、MOTU和Adobe的獨特策略

    儘管蘋果和 Avid 一直在追求“垂直整合”， Steinberg、MOTU 和 Adob​​​​e 通過“加強其專業領域”而不斷壯大。

    ■ Steinberg：隸屬於雅馬哈（2004年）

    即使收購後，方向也沒有改變，技術發展仍在繼續。

    • Nuendo進軍後期製作市場
    • 使用 VST3 (2006) 擴展表現力
    • Cubase 作為作曲 DAW 保持全球份額

    ■ MOTU：作為影視作曲家的標準而建立

    • DP5/6時代，視頻同步的準確性進一步增強。
    • 它僅適用於 Mac，這一事實使其與視頻系統兼容。
    • 被經常使用軟合成器的音樂作曲家廣泛採用

    ■ Adob​​​​e：Cool Edit → 與 Audition 集成廣播編輯

    收購之後，我們將重點關注與Premiere的合作。

    • 視頻編輯器可以直接調整音頻
    • 成為廣播電台和播客的標準
    • 它更像是一個“編輯工具”，而不是混音器/作曲 DAW。

    這裡是 DAW 市場 作曲/錄音/編輯/視頻/廣播將根據用途進行區分。

    6. 2010年代：原生化和操作系統/硬件集成的時代

    在此期間，DAW 憑藉依賴於 CPU 的本機處理成為主流。

    ■ 蘋果：Logic Pro X (2013)

    • 界面刷新
    • 大量柔和的聲源
    • 價格更低（10,000日元以下）
    • 優化以最大限度地提高 Mac 性能

    ■ Avid：遷移到 HDX 和 AAX

    • 集成 DSP 和本機
    • Pro Tools 仍然是工作室標準
    • 對於大規模的電影混音，除了Pro Tools之外幾乎別無選擇。

    ■ Steinberg、MOTU、Adobe也持續更新

    • Steinberg 更新了 VST3 引擎
    • MOTU 通過 DP9/10 進入現代工作流程
    • Adobe 的 Audition 滲透率歸因於對播客的需求不斷增加

    到了這個時候，“DAW”這個詞已經完全普及了， 加上家庭錄音文化的擴張，整體市場將迅速擴大。

    7. 2020-2025：Apple Silicon 和訂閱時代

    在過去的幾年裡，DAW 的歷史又發生了一次重大變化。

    ■ 蘋果：Apple Silicon 的速度快得驚人

    • Logic Pro 針對 ARM 架構進行了優化
    • 使用 iPad 版 Logic 引領移動製作 (2023)
    • 低延遲、高效率備受好評

    ■ Avid：Pro Tools 訂閱遷移

    • 從永久許可到訂閱
    • 杜比全景聲 (Dolby Atmos) 工作流程成為標準
    • 繼續在電影音響市場佔據主導地位

    ■ Steinberg：強化Nuendo高端領域

    • 兼容杜比全景聲
    • 提高遊戲音頻行業的影響力

    ■ MOTU：DP11進入新時代

    • 合成 - 增強視頻同步工作流程
    • 器樂作曲家和伴奏作家的持續支持

    ■ Adob​​​​e：發行和廣播需求增加

    • 與 Premiere 的集成優勢
    • YouTube/Podcast 時代的強勢

    8. 6家公司比較：DAW歷史上“角色”的差異

    支持 DAW 歷史的六家公司有著明確的角色分工。

    公司 領先領域 意義
    蘋果（邏輯） 作曲/家庭錄音 與Mac集成，拓展軟件音源文化
    Avid（專業工具） 電影/聲音/錄音室 專用DSP、編輯能力、視頻聯動
    斯坦伯格 標準化 (VST)/綜合 DAW 影響所有 DAW 的技術基礎
    摩圖 視頻同步/伴奏作曲 電影評分標準
    Adobe 廣播、編輯和發行 用於與 Premiere 集成進行視頻製作
    emagic / Digidesign 收購前的起源 Mac獨占使用/電影聲音基礎的形成

    9.年表（美人魚）

    ■ DAW 歷史概述（1980-2025）

    timeline title DAW史総覧：Apple / Avid / Steinberg / MOTU / Adobe 1984 : Steinberg & Digidesign 設立 1985 : MOTU Performer 1989 : Cubase 1.0 1991 : Pro Tools 1.0 1993 : Logic 1.0 1996 : VST 1.0 1999 : VSTi 2002 : Apple が emagic を買収（Logic Mac専用に） 2003 : Adobe が Cool Edit を買収（Audition誕生） 2004 : Steinberg が Yamaha 傘下へ 2010 : Digidesign ブランド終了 2013 : Logic Pro X 2020 : Pro Tools サブスク中心へ 2023 : Logic Pro for iPad

    10、結論：DAW的歷史是一個“企業戰略×技術標準×市場需求”交叉的龐大故事

    Apple 收購了 emagic 和 Avid 集成的 Digidesign。 DAW 市場沿著 OS x 視頻製作 x 工作室基礎設施 的軸發生了顯著變化。 另一方面，斯坦伯格創建了一個名為 VST 的歷史標準， MOTU已經掌握了視頻同步， Adobe 通過與廣播和視頻編輯的集成創建了自己的市場。

    儘管六家公司相互競爭， 結果，他們共同塑造了“音樂製作本身的文化”。

    2025年後，人工智能剪輯、雲協作、移動製作拓展等。 DAW 的未來將呈現出一個新的維度。 然而，這的根源在於這裡提到的六家公司的足跡。
