    【專欄】Waajed——持續更新底特律譜系的創造者

    Column zh-hant 10s Deep House House
    28 Nov 2025

    介紹

    文：mmr｜主題：瓦吉德的生平和作品，以及他在底特律家譜中的角色

    在討論底特律的音樂史時，焦點往往集中在早期電子音樂的巨頭上。然而，自2000年代以來，有許多藝術家“重新整合”了這座城市自己的黑人音樂血統——嘻哈、福音、靈魂樂、浩室、爵士樂和電子音樂——並將其昇華為新的表達形式。

    瓦吉德是這件事的中心人物。 他的職業生涯始於貧民窟村，隨後他與 Platinum Pied Pipers (PPP)、Tiny Hearts 和他自己的廠牌 Dirt Tech Reck 繼續將底特律的音樂美學功能打造為“連接未來的平台”。

    自 2010 年代末以來，他一直在創作一系列更直接遵循底特律 house/techno 潮流的作品，同時也將黑人根源的表達帶到了最前沿。 特別是《Shango EP》（2017）和他們當前的代表作之一《Mother EP》（2018）被定位為在繼承底特律傳統的同時強烈呈現俱樂部音樂的靈性和社區性質的作品。

    1.早期生活和背景：底特律，“起源之城”

    Waajeed（真名羅伯特·奧布萊恩特）於 20 世紀 70 年代末出生於底特律。 藝術家的名字源自阿拉伯語，取自意思是“尋找”或“發現”的單詞。

    他從小就在底特律靈魂/福音文化影響的環境中長大，後來接觸了貧民窟村。 20 世紀 90 年代末和 2000 年代初，底特律正在為新靈魂樂/嘻哈音樂制定新標準，以 J Dilla 為代表。瓦吉德處於這一領域的前沿，在擔任攝影師的同時，他轉向音樂製作。

    ● 與貧民窟村的關係

    • 作為集團外圍人員參與
    • 與會員T3和Baatin互動
    • 直接受到 J Dilla 創造力的影響

    瓦吉德在這裡吸收的不僅僅是節拍製作。 態度是“將黑人音樂的傳承作為現代時代的‘再創造’。”

    2. PPP（白金花衣風笛手）時期：嘻哈與靈魂樂之間的新連接點

    2000 年代初，Waajeed 和 Saadiq 組建了 Platinum Pied Pipers (PPP)。 這是一個有機融合嘻哈、R&B、靈魂樂和節拍音樂的項目，在底特律獲得了全世界的讚譽。

    2005年“三P”

    • PPP首張專輯
    • 由喬治婭·安妮·莫德羅、蒂姆貝·洛克哈特、薩迪克等主演
    • 它被譽為當時新靈魂樂和嘻哈背景下“最激進、最溫暖的作品”之一。
    • 這是一部讓底特律黑人音樂美學享譽國際的重要作品。

    在此期間，瓦吉德模糊了採樣和現場音樂之間的界限，並致力於創造一種非俱樂部音樂的“未來靈魂”風格。 **

    PPP 是 2000 年代底特律最具標誌性的項目之一，也是 Waajeed 職業生涯的基礎。

    3. Dirt Tech Reck 品牌的成立和使命

    2012 年，Waajeed 推出了自己的品牌 Dirt Tech Reck。 正如“Dirt”、“Tech”和“Reck”這三個詞所表明的那樣，

    • 街頭文化的污垢
    • 科技的未來
    • 記錄文化的RECK（記錄）

    目的是將這些整合起來。

    Dirt Tech Reck 的獨特之處在於它不僅僅是經營一個廠牌。 它充當黑人藝術家文化自治的平台。

    許多版本都具有以下元素：

    • 根植於底特律的聲音
    • 黑人社區敘事和社會背景
    • 舞池與靈性之間的聯繫
    • 跨越嘻哈音樂、浩室音樂和科技音樂的結構

    從這裡開始，瓦吉德將進入與PPP時期不同的領域。 **將焦點轉移到俱樂部音樂的背景上。 **

    4. “Shango EP”（2017）：成為轉折點的儀式浩室/電子音樂

    Shango EP 於 2017 年發行，是 Waajeed 職業生涯的一個重要轉折點。

    標題“Shango”是西非傳統宗教中出現的神靈，也是將非洲僑民精神與黑人音樂歷史聯繫起來的象徵。

    在這張 EP 中，瓦吉德：

    • 底特律 house/techno 節拍結構
    • 非洲打擊樂
    • 黑色的心靈呼喚
    • 原始聲音和節奏的儀式

    通過融合，他明確引導樂隊“代表 house/techno 回歸黑藝術精神”。

    這部作品不是PPP和嘻哈時代的延伸，而是呈現了底特律音樂的新階段，並在全世界獲得了高度評價。

    5. 《Mother EP》（2018）：靈性、夜店和黑人的融合

    《Mother EP》(2018) 被廣泛認為是 Waajeed 的傑作。 這部作品的主題不僅僅是 house/techno，而是作為黑人社區記憶和靈性檔案的俱樂部音樂。

    特徵

    • 底特律福音影響力
    • 深情的歌聲
    • 沉重而動感的節拍
    • 非洲未來主義聲音圖像
    • 重新思考象徵“母性”的社區文化

    與 PPP 和嘻哈時代的節拍製作相比，它具有更加線性和舞池導向的結構，但它也具有明顯的黑人根源的情感表達。 它所處的位置可以被稱為“Dirt Tech Rec 的美學目的地”。

    6。製作態度：重新定義底特律的實踐者

    總結Waajeed的製作方式，主要有以下三點。

    ①繼承並再造底特律的遺產

    • J·迪拉的影響
    • 福音/靈魂傳統
    • 嘻哈美學
    • 科技的未來

    它們並沒有被分割成碎片，而是被整合為“底特律市本身的音樂記憶”。

    ② 生產紮根社區

    在 Dirt Tech Reck 時期之後，瓦吉德開始不再將音樂視為“個人藝術”，而是將音樂視為“個人藝術”。 **人們強烈希望使其成為當地社區的文化設施。 **

    • 參與教育項目
    • 底特律培養年輕創作者
    • 嘗試將社區的歷史融入音樂中

    ③ 重新連接黑人靈性

    特別是自從《Shango EP》以來，非洲僑民的精神已經成為一個重要的元素。

    • 呼叫和響應
    • 儀式打擊樂
    • 福音派表達
    • 恍惚節奏

    這些表演代表了“house/techno 的起源”。

    7.作品主要特點及音樂性分析

    ■ 節奏與打擊樂

    Waajeed的節奏與“硬度”和“原始搖擺”並存， 他們更看重黑人舞曲的節奏，而不是電子音樂的極簡主義。

    ■ 人聲和合唱

    自PPP時期以來，出現了很多福音般的聲音處理。 這在《Mother》中表現得更為明顯，賦予了俱樂部曲目情感的深度。

    ■ 現場聲音和電子聲音的混合

    自然混合合成器、pad、電鋼琴、現場鼓、打擊樂等的技巧也是嘻哈時代“耳朵訓練”的結果。

    11.瓦吉德的文化意義

    瓦吉德是一位罕見的藝術家，他不將底特律的音樂文化視為一種“流派”，而是將其視為歷史和公共記憶的連續體。

    • 嘻哈 → 靈魂 → 福音 → House → 電子音樂 從他一手體現這一血統的意義上來說，他體現了典型的“底特律創造者”的形象。

    而且，這次創作是為了創新，而不是懷舊。 從《Shango EP》到《Mother》的作品分別是 它將繼續被稱為同時審視黑人音樂的根源和未來的俱樂部音樂。

    結束

    瓦吉德的職業生涯就是通過將底特律市的“歷史層次”與當代俱樂部文化相結合，繼續重新詮釋它們。 不僅個人作品具有吸引力； 作為將城市記憶和黑人靈性轉化為音樂的方法論，它也很重要，它的存在將繼續成為支持底特律音樂演變的象徵。
