##  Command Records和“聲音設計”——有什麼創新之處？

文字：mmr |主題：仔細追溯將立體聲時代的“示範唱片”提升到藝術層面的伊諾克·賴特（Enoch Wright）的歷史、代表作品、相關藝術家、軼事和影響，以及他創立的公司Command Records至今的發展歷程。


在20世紀50年代和1960年代，音響愛好者中有人將他們的定位從“要聽的唱片”轉變為“要聽的唱片”。指揮/製作人Enoch Light於1959年創立了追求高品質聲音的Command Records唱片公司，並以其獨特的藝術作品和尖端的錄音技術成為“hi-fi文化”的象徵。本文總結了整個故事，包括其歷史背景、代表作、與托尼·莫托拉的關係以及今天的評價。

### “高保真休息室/太空時代流行音樂”

Command Records明確定位為“音頻示範唱片”市場，以音質為最高優先級來選擇、編曲和錄製歌曲。賴特引進了多麥克風錄音和35mm膠片母帶錄音等最新技術，並以強烈的聲場意識（立體聲圖像）進行製作。由此產生了一批強調“平移效果”和打擊樂器本土化的專輯，並確立了有別於一般流行音樂和爵士樂的“hi-fi休息室/太空時代流行音樂”流派。

---


### 軼事、傳說、情節

- **摺頁和圖形美學**

Command 的唱片封面視覺效果很強，有時封面是抽像畫家約瑟夫·阿爾伯斯 (Josef Albers) 的作品。 CD 的設計超越了單純的演示光盤，成為人們想要擁有的東西，刺激了隨後的收藏文化。


- **聲稱：“當你第一次聽到它時，你會感到驚訝”**

Stereo 35/MM的內頁註釋和促銷使用了挑釁性的短語，例如“第一次聽到它時你會感到驚訝”，以引起聽眾的興趣。這是基於技術信心的成功營銷的例子。


- **商業與藝術的衝突**

接近“演示”版本的音質受到發燒友的歡迎，但也有人質疑其作為流行歌曲的普遍性。儘管如此，在 Chart 上還是取得了一些成功（Stereo 35/MM 等），Wright 是少數能夠使“技術吸引力”在商業上可行的製作人之一。

### 代表系列及專輯

有說服力的打擊樂系列（1959年首次出版）
該系列旨在成為前景為打擊樂器的“立體聲展示品”。它的選曲和編排在圈外人中也產生了很大的影響，因此獲得了很高的人氣。封面的圖形和摺頁的解釋在當時也是創新的。


挑釁打擊樂系列
《說服力》的姐妹系列。雖然理念相似，但有些作品在編曲和獨奏表現上有所改進，並在Billboard排行榜上取得了較高的排名。


立體聲 35/MM (1961)
這張專輯是在卡內基音樂廳使用 35 毫米母帶錄製的。它在發布後很長一段時間都保持在排行榜的首位，是同時實現技術吸引力和商業成功的典型例子。

### 參與的音樂家/製作的藝術家

除了親自指揮之外，Enoch Light 還為每張專輯聘請了頂級音樂家（例如 Doc Severinsen、Phil Bodner 等）。他還發行了特里·斯奈德、托尼·莫托拉和迪克·海曼等表演者/編曲家的專輯，展現了他作為製作人的多才多藝。作為一名錄音工程師，他與 Bob Fine 等人合作，進行了許多聲學實驗。

## 托尼·莫托拉和伊諾克·萊特之間的關係

托尼·莫托拉 (Tony Mottola) 和伊諾克·萊特 (Enoch Light) 是 Command Records 的核心關係人。

在 20 世紀 50 年代和 1960 年代的高保真/立體聲錄音文化中，**Enoch Light** 被稱為“聲音建築師”。他的得力助手是爵士樂/錄音室吉他手 **托尼·莫托拉**，一直支持他的表演和獨奏工作。他們在 Command Records 的關係不僅僅是製作人和歌手。

### 托尼·莫托拉是誰？

美國爵士樂/錄音室吉他手 (1918–2004)

他活躍於 NBC 管弦樂團和電視節目的演出中，並且是 20 世紀 50 年代至 1970 年代紐約最忙碌的吉他手之一。

他演奏的音樂風格非常廣泛，不僅包括爵士樂，還包括輕音樂、拉丁音樂、流行音樂和電影音樂。

---

### 與伊諾克·萊特的相遇和關係

- **大獎 → Command Records 期間的常任吉他手**

當伊諾克·萊特 (Enoch Light) 創辦 Grand Award 和 Command 時，他聚集了紐約頂尖的會議人才。其中，固定吉他手是托尼·莫托拉（Tony Mottola）。

他還在“Persuasive Percussion”和“Provocative Percussion”系列中扮演了重要角色。

在拉丁節奏和立體聲定位的實驗中，吉他作為一種可以輕鬆分配左右的樂器很有用，莫托拉靈活的演奏得到了很好的利用。

- **個人專輯發行（來自 Command）**

Command Records 還發行了莫托拉的許多個人作品。

其中《羅馬吉他》（1960）系列尤為著名，是一首高保真的改編拉丁和意大利民歌的流行作品。

此後，他的大部分傑作如《Roman Guitar Vol. 2》和《Roman Guitar Vol. 3》都是在Enoch Light的指導下製作的。

莫托拉是支持指揮部“休息室/輕鬆聆聽路線”的支柱之一。

---

### 編曲與錄音風格的信任關係

- **Command Records 常規吉他手**

他幾乎總是參與Light設立的Grand Award~Command的錄音。
負責《Persuasive Percussion》系列等作品中的吉他本土化實驗。

- **發行個人專輯**

Command開發了“羅馬吉他”系列，使拉丁和意大利音樂變得高保真。它成為舉世聞名的傑作。

- **在聲學實驗中的作用**

將吉他放置在立體聲左側和右側的定位實驗

- **與節奏部分互動的單音符主音**

強調混響和混響處理的效果

→ 莫托拉靈活的演奏將萊特的願景變為現實。

### 現狀及影響

Command Records 本身實際上在 1970 年左右就結束了，但他們的作品仍然通過重新發行、高質量重新製作以及為發燒友重新發行而流傳。有說服力/挑釁性的歌曲和夾克繼續在“中世紀設計”和單身漢文化的背景下被引用，也是採樣和混音的主題。它對現代聆聽文化（復古音頻愛好、虛擬房間模擬等）的影響相當大。


---

### 摘要：以諾之光的遺產

Enoch Light是一個不僅追求“好聲音”的人，還設計了一種叫做唱片的物體，重新定義了立體聲時代的“聆聽體驗”。如今，Command Records 的作品仍在音頻史、設計史和流行文化的背景下被重新評估。創造你第一次聽音樂時感受到的“驚喜”是賴特的真正精髓，他的作品至今仍然保留著這種魔力。

此外，伊諾克·萊特和托尼·莫托拉之間的關係不僅僅是“製作人和吉他手”。 Command Records 將 Light 的錄音美學與 Mottola 靈活多變的吉他演奏相結合，成為太空時代的流行偶像，繼續吸引著世界各地的音響發燒友。
